# Pasta de icones

Adicione aqui os arquivos obrigatorios para publicacao na Chrome Web Store:

- `icon-16.png` (16x16)
- `icon-48.png` (48x48)
- `icon-128.png` (128x128)

Quando voce adicionar os arquivos, inclua no `public/manifest.json`:

```json
"icons": {
  "16": "icons/icon-16.png",
  "48": "icons/icon-48.png",
  "128": "icons/icon-128.png"
}
```
